O que é HTML?

HTML é uma das linguagens de marcação mais conhecidas e utiliza um formato simples de codigo, que pode ser gerado sem a ajuda de aplicativos especias

Para criar um documento HTML, tudo o que você precisa é de um editor de texto ASCII como o NotePad (Bloco de Notas), por exemplo. Se você precisar verificar a formatação do documento durante a criação, pode usar qualquer navegador, pois todos eles são capazes de interpretar o codigo.
