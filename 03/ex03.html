Para criar um documento HTML é
necessário seguir os passos abaixo.

  1. Acessar o bloco de notas
  2. Indicar através da tag
    HTML o tipo de documento criado.

Esses são os primeiros passos para se criar uma pagina web.

Segue abaixo lista de materiais necessarios:

  • Leads
  • Conectores
  • Chave de fenda Philips
  • Alicate

Somente começe a instalação, se possuir todos os materias.

  1. Orange
    • Tela sensivel ao toque de 13"
    • 3 entradas USB
  2. Lemon
    • Tela sensivel ao toque de 15"
    • 2 entradas USB

Após a análise, o grupo deverá escolher
a melhor opção para trabalho a ser realizado.
